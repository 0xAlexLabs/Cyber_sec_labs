## Заметки по прохождению OverTheWire Bandit.

Bandit — это обучающий wargame от OverTheWire, предназначенный для начинающих. Он помогает освоить базовые навыки работы с Linux и командной строкой, необходимые для прохождения более сложных заданий.

Игра состоит из уровней: вы начинаете с Level 0 и, проходя их последовательно, получаете доступ к следующим. Каждый уровень требует применения базовых команд и поиска решений.
