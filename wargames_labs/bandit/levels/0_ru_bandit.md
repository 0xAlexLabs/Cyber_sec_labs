## 🧠 Цель уровня
Подключиться к серверу Bandit по SSH.

---

## 📌 Дано
- Хост: bandit.labs.overthewire.org
- Порт: 2220
- Логин: bandit0
- Пароль: bandit0

---

## 💻 Решение

```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
```

---

## 📖 Объяснение

Команда:

```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
```

Разбор:

- `ssh` — программа (Secure Shell) для удалённого подключения к серверу  
- `bandit0@bandit.labs.overthewire.org` — указывает пользователя (`bandit0`) и адрес сервера  
- `-p 2220` — подключение к нестандартному порту (по умолчанию используется порт 22)  

Как это работает:
- устанавливается защищённое соединение с сервером  
- происходит попытка входа под пользователем `bandit0`  
- после этого система запрашивает пароль  

При первом подключении:
- нужно подтвердить доверие к серверу (`yes`)  
- затем ввести пароль `bandit0`  

Важно:
- при вводе пароля символы в терминале не отображаются — это нормально  

После успешного входа вы попадаете в удалённую командную строку (shell).
