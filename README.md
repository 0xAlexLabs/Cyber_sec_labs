# 🛡️ Лабораторные работы по веб-безопасности

## 📌 О проекте

Данный репозиторий содержит мои решения и разборы лабораторных работ по
веб-безопасности, в первую очередь с платформы PortSwigger Web Security
Academy, Hack the box и возможно другие в процессе обучения.

Основная цель - системно развивать практические навыки в области
пентеста, разбирая реальные сценарии уязвимостей и способы их
эксплуатации.

------------------------------------------------------------------------

## 🎯 Цели обучения

-   Построить прочную базу знаний в веб-безопасности
-   Научиться выявлять и эксплуатировать уязвимости
-   Понимать реальные векторы атак и способы защиты
-   Вести структурированную документацию своего прогресса

------------------------------------------------------------------------

## 📝 Формат разборов

Каждая лабораторная работа включает:

-   📖 Краткое описание
-   🔍 Анализ уязвимости
-   ⚙️ Пошаговую эксплуатацию
-   💡 Ключевые выводы
-   🛠️ Используемые инструменты и техники

------------------------------------------------------------------------

## 🚀 Прогресс

Проект находится в процессе развития - новые лабораторные работы
добавляются по мере прохождения.

------------------------------------------------------------------------

## ⚠️ Дисклеймер

Все действия, описанные в данном репозитории, выполняются исключительно
в учебных и контролируемых средах (например, PortSwigger labs и HTB virtual machines).

------------------------------------------------------------------------

## 📫 Примечание

Этот репозиторий - часть моего пути в пентест и offensive security.

Если вы тоже изучаете безопасность - можете использовать материалы для
обучения.
