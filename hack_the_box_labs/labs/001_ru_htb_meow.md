## 🎯 Цель

Получить доступ к удалённой машине и извлечь флаг, используя базовые техники разведки (enumeration) и эксплуатации.

---

## 🧠 Ключевая идея

Машина уязвима из-за:

- открытого сервиса Telnet  
- отсутствия пароля у критической учетной записи (root)

Это пример misconfiguration (ошибочной конфигурации).

---

## 🔍 Ход решения

### Шаг 1 — Проверка доступности

```
ping <TARGET_IP>
```

📌 Убеждаемся, что машина доступна.

---

### Шаг 2 — Сканирование портов

```
nmap -sV <TARGET_IP>
```

📌 Результат:
- открыт порт 23/tcp  
- сервис: telnet  

---

### Шаг 3 — Подключение к сервису

```
telnet <TARGET_IP>
```

📌 Получаем приглашение для ввода логина.

---

### Шаг 4 — Подбор учетных данных

Пробуем:
- admin  
- administrator  
- root  

📌 Успешный вход:
- логин: root  
- пароль: (пустой)

---

### Шаг 5 — Поиск флага

```
ls
cat flag.txt
```

📌 Получаем флаг.

---

## 💥 Уязвимость

- Telnet (без шифрования)  
- root без пароля  
- отсутствие защиты аутентификации  

👉 Полная компрометация системы
